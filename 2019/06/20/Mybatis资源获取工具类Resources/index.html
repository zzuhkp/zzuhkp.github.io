Mybatis资源获取工具类Resources

发表于 2019-06-20 | 分类于 Mybatis源码阅读 |
Mybatis工具类Resources在获取Mybatis配置文件，Mybatis xml dtd文件中都有使用到，本文对Resources工具类进行源码分析。

一 Resources包含的方法

Resources提供各种从类路径获取资源的方法，主要依赖内部的classLoaderWrapper完成资源的查找，提供的方法主要有以下几种

  • getResourceAsFile
  • getResourceAsProperties
  • getResourceAsReader
  • getResourceAsStream
  • getResourceURL
  • getUrlAsProperties
  • getUrlAsReader
  • getUrlAsStream

这些方法主要是返回的类型不同，getResourceXXX方法最终都调用了classLoaderWrapper.getResourceAsURL或classLoaderWrapper.getResourceAsStream方法。
